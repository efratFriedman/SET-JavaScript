מטרת המשחק

מטרת המשחק היא לזהות set - סדרה מנצחת של 3 קלפים מתוך 12 קלפים שמונחים על השולחן

חוקי המשחק

1. ארבעה מאפיינים מבחינים בין 81 קלפי המשחק השונים:

א- צורה בכל קלף מופיעים צורות דמויות אליפסה, יהלומים, או שרבוטים
ב- צבע הצורות יכולות להופיע באחד משלושה הצבעים: אדום, ירוק או סגול
ג- מספר בכל קלף עשויים להופיע אחד, שניים או שלושה סמלים (זהים)
ד- מילוי הסמלים על הקלף יכולים להיות מלאים, מקווקווים או ללא מילוי צבע

2. set מורכב משלישיה של קלפים הממלאים את התנאי הבא:
כאשר אנו בוחנים כל אחד מארבעת המאפינים- צורה, צבע, מספר, ומילוי כל אחד בתורו המאפיין הזה זהה בשלושת הקלפים או שונה בכל אחד מהם
במילים אחרות הצורה צריכה להיות זהה בשלושת הקלפים או שונה בכל אחד מהם וכך גם לגביי בצורה המספר והמילוי

כלל הזהב
אם רק שניים מתוך שלושת הקלפים הם בעלי תכונה זהה כלשהי- זהו אינו set חוקי במשחק set כל אחד מארבעת המאפינים צריך להיות זהה לחלוטין או שונה לחלוטין בכל שלושת הקלפים
